# Node Palette Specification (Modified)

## MODIFIED Requirements

### 需求:节点调色板必须应用深色主题

左侧节点调色板必须使用深色背景和霓虹边框发光效果。

#### 场景:深色主题调色板背景
- **当** 用户使用深色主题
- **那么** 调色板必须显示：
  - 背景颜色：#1E293B
  - 右侧边框：霓虹蓝发光效果

#### 场景:浅色主题调色板背景
- **当** 用户使用浅色主题
- **那么** 调色板必须显示：
  - 背景颜色：#F8FAFC
  - 右侧边框：蓝色实线

### 需求:节点卡片必须有霓虹发光效果

调色板中的节点卡片必须有霓虹发光的悬停效果。

#### 场景:节点卡片悬停效果
- **当** 用户悬停在节点卡片上
- **那么** 卡片必须显示：
  - 霓虹发光边框
  - 发光强度根据节点类型变化

### 需求:节点卡片文字必须有足够对比度

节点卡片中的文字必须与背景有足够的对比度。

#### 场景:节点卡片文字对比度验证
- **当** 使用 Chrome DevTools Accessibility 面板检查
- **那么** 所有文字对比度必须大于 4.5:1

### 需求:节点卡片必须有拖拽视觉反馈

节点卡片在拖拽时必须有明显的视觉反馈。

#### 场景:拖拽开始视觉反馈
- **当** 用户开始拖拽节点卡片
- **那么** 卡片必须显示拖拽状态（如透明度降低、缩放变化）

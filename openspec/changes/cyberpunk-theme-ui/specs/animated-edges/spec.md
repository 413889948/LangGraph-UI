# Animated Edges Specification

## ADDED Requirements

### 需求:连接线必须显示流动光效动画

节点间的连接线必须显示流动光效动画，使用 CSS stroke-dasharray 和 stroke-dashoffset 实现。

#### 场景:直接边动画
- **当** 用户创建两个节点之间的直接连接
- **那么** 连接线必须显示流动光效动画

#### 场景:条件边动画
- **当** 用户创建条件边连接
- **那么** 连接线必须显示流动光效动画，并显示条件标签

### 需求:连接线颜色必须使用渐变

连接线颜色必须从源节点颜色渐变到目标节点颜色。

#### 场景:Function 到 Tool 连接线颜色
- **当** Function 节点连接到 Tool 节点
- **那么** 连接线必须从霓虹蓝渐变到霓虹绿

#### 场景:Tool 到 Subgraph 连接线颜色
- **当** Tool 节点连接到 Subgraph 节点
- **那么** 连接线必须从霓虹绿渐变到霓虹紫

### 需求:选中连接线必须有视觉反馈

当连接线被选中时，必须有明显的视觉反馈。

#### 场景:连接线选中效果
- **当** 用户选中一条连接线
- **那么** 连接线必须显示：
  - 发光效果
  - 线条宽度增加

### 需求:连接线动画必须流畅

连接线动画帧率必须保持在 50 FPS 以上。

#### 场景:动画性能验证
- **当** 使用 Chrome DevTools Performance 面板测试
- **那么** 动画帧率必须大于 50 FPS

### 需求:连接线动画必须尊重 prefers-reduced-motion

当用户偏好减少动画时，连接线动画必须减少或禁用。

#### 场景:用户偏好减少动画
- **当** 用户系统设置 `prefers-reduced-motion: reduce`
- **那么** 连接线流动动画必须禁用或减慢

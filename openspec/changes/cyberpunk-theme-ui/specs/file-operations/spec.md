# File Operations Specification (Modified)

## MODIFIED Requirements

### 需求:文件操作按钮必须有霓虹发光效果

文件操作按钮必须应用霓虹发光效果，包括新建、加载、保存、导出代码按钮。

#### 场景:主要按钮发光效果
- **当** 用户查看保存或导出按钮
- **那么** 按钮必须显示霓虹蓝发光效果

#### 场景:次要按钮样式
- **当** 用户查看新建或加载按钮
- **那么** 按钮必须显示霓虹蓝边框发光效果

### 需求:按钮悬停效果必须增强发光

按钮在悬停状态时必须增强发光效果。

#### 场景:按钮悬停发光增强
- **当** 用户悬停在按钮上
- **那么** 发光强度必须增加 50%

### 需求:按钮图标必须使用 SVG

按钮必须使用 SVG 图标而非 emoji。

#### 场景:按钮图标验证
- **当** 用户查看按钮
- **那么** 按钮图标必须是 SVG 元素，不能使用 emoji 字符

### 需求:禁用按钮必须有视觉区分

禁用状态的按钮必须有明显的视觉区分。

#### 场景:按钮禁用状态
- **当** 按钮处于禁用状态
- **那么** 按钮必须显示：
  - 降低的透明度
  - 移除发光效果
  - 禁用鼠标指针

### 需求:按钮必须有可访问性支持

按钮必须支持键盘操作和屏幕阅读器。

#### 场景:按钮焦点状态
- **当** 用户使用 Tab 键导航到按钮
- **那么** 按钮必须显示清晰的焦点环

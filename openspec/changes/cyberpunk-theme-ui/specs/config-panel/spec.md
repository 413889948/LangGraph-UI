# Config Panel Specification (Modified)

## MODIFIED Requirements

### 需求:配置面板必须应用深色主题

右侧配置面板必须使用深色背景和霓虹 Tab 激活效果。

#### 场景:深色主题配置面板背景
- **当** 用户使用深色主题
- **那么** 配置面板必须显示：
  - 背景颜色：#1E293B
  - 左侧边框：霓虹蓝发光效果

### 需求:Tab 切换必须有霓虹发光激活效果

Tab 按钮在激活状态时必须显示霓虹发光效果。

#### 场景:Tab 激活状态
- **当** 用户点击 Tab 按钮
- **那么** 激活的 Tab 必须显示：
  - 霓虹蓝边框颜色（#00D4FF）
  - 发光效果

#### 场景:Tab 非激活状态
- **当** Tab 未被选中
- **那么** Tab 必须显示灰色文字和无发光效果

### 需求:表单控件必须有深色主题样式

输入框、下拉框等表单控件必须应用深色主题样式。

#### 场景:输入框深色主题样式
- **当** 用户使用深色主题查看输入框
- **那么** 输入框必须显示：
  - 背景颜色：#334155
  - 边框颜色：#475569
  - 文字颜色：#F8FAFC

#### 场景:输入框焦点效果
- **当** 用户聚焦输入框
- **那么** 输入框必须显示霓虹蓝边框发光效果

### 需求:表单控件文字必须有足够对比度

表单控件中的文字必须与背景有足够的对比度。

#### 场景:表单控件文字对比度验证
- **当** 使用 Chrome DevTools Accessibility 面板检查
- **那么** 所有文字对比度必须大于 4.5:1

### 需求:参数列表必须有深色主题样式

参数列表项必须应用深色主题样式。

#### 场景:参数项深色主题样式
- **当** 用户查看参数列表
- **那么** 每个参数项必须显示深色背景和清晰的边框

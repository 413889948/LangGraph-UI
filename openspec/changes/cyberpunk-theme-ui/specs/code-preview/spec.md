# Code Preview Specification (Modified)

## MODIFIED Requirements

### 需求:代码预览必须应用深色主题

代码预览区域必须使用深色背景和代码高亮。

#### 场景:代码预览深色主题背景
- **当** 用户使用深色主题查看代码预览
- **那么** 代码预览区域必须显示：
  - 背景颜色：#1E1E1E 或类似深色
  - 代码文字颜色：#D4D4D4

### 需求:代码预览头部必须有深色主题样式

代码预览的头部区域必须应用深色主题样式。

#### 场景:代码预览头部样式
- **当** 用户查看代码预览头部
- **那么** 头部必须显示深色背景和霓虹发光标题

### 需求:复制按钮必须有霓虹发光效果

复制代码按钮必须有霓虹发光效果。

#### 场景:复制按钮悬停效果
- **当** 用户悬停在复制按钮上
- **那么** 按钮必须显示霓虹蓝发光效果

### 需求:验证状态必须有清晰的视觉区分

验证状态（有效/无效）必须有清晰的视觉区分。

#### 场景:有效状态显示
- **当** 生成的代码通过验证
- **那么** 状态指示器必须显示绿色（#22C55E）和成功图标

#### 场景:无效状态显示
- **当** 生成的代码验证失败
- **那么** 状态指示器必须显示红色（#EF4444）和错误图标

### 需求:错误信息必须有清晰的显示

验证错误信息必须有清晰的显示和足够的对比度。

#### 场景:错误信息显示
- **当** 代码存在验证错误
- **那么** 错误列表必须显示：
  - 红色背景高亮
  - 清晰的错误描述
  - 文字对比度 > 4.5:1

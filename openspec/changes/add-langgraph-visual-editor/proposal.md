## 为什么

当前开发者在构建 LangGraph 应用时缺少可视化工具，只能通过手写代码定义图结构、节点和边，导致学习曲线陡峭、调试困难、协作效率低。可视化编辑器能够降低认知负担，提升开发体验和迭代速度。

## 变更内容

- **新增可视化图编辑器**：基于 React Flow 实现节点式工作流编辑界面，支持拖拽创建节点、手动连接边、可视化配置状态图
- **新增节点配置面板**：提供节点属性编辑界面，包括节点类型、输入输出、处理逻辑配置
- **新增状态/Schema 编辑器**：可视化编辑 LangGraph 状态定义，支持字段类型、reducer 配置
- **新增图导出与代码生成**：将可视化图导出为 LangGraph Python 代码，支持 StateGraph 定义、节点注册、边连接逻辑
- **新增项目持久化**：支持保存和加载可视化图项目文件，便于版本管理和协作

## 功能 (Capabilities)

### 新增功能
- `graph-visualization`: 可视化 LangGraph 图结构，支持节点拖拽、边连接、画布缩放平移
- `node-configuration`: 节点属性配置，包括节点类型选择、参数设置、输入输出定义
- `state-schema-editor`: 状态 Schema 可视化编辑，支持字段添加/删除、类型选择、reducer 配置
- `code-generation`: 从可视化图生成 LangGraph Python 代码，导出可执行的 StateGraph 定义
- `project-persistence`: 项目文件保存与加载，支持 JSON 格式存储图结构和配置

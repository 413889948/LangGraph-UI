## 新增需求

### 需求:用户可以在画布上创建和管理节点
系统必须允许用户通过拖拽方式在画布上创建节点，并支持节点的移动、选择和删除操作。

#### 场景:通过拖拽创建节点
- **当** 用户从节点工具栏拖拽一个节点类型到画布上
- **那么** 系统在鼠标释放位置创建一个该类型的新节点，并显示默认配置

#### 场景:移动节点位置
- **当** 用户拖拽画布上的节点到新的位置
- **那么** 节点移动到新的位置并保持其所有配置不变

#### 场景:选择节点
- **当** 用户点击画布上的一个节点
- **那么** 该节点被高亮显示，表示已被选中

#### 场景:删除节点
- **当** 用户选中一个节点并按下删除键或点击删除按钮
- **那么** 该节点及其连接的所有边被从画布上移除

### 需求:用户可以创建和编辑节点之间的边
系统必须支持用户通过手动连接的方式创建节点之间的边，并支持边的删除操作。

#### 场景:创建直接边
- **当** 用户从一个节点的输出锚点拖拽到另一个节点的输入锚点
- **那么** 系统在两个节点之间创建一条直接边

#### 场景:删除边
- **当** 用户选中一条边并按下删除键或点击删除按钮
- **那么** 该边从画布上被移除，不影响连接的节点

### 需求:画布支持缩放和平移导航
系统必须提供画布的缩放和平移功能，以便用户可以查看大型图的不同部分。

#### 场景:缩放画布
- **当** 用户使用鼠标滚轮或缩放控件
- **那么** 画布按用户操作放大或缩小，保持节点相对位置不变

#### 场景:平移画布
- **当** 用户在空白区域拖拽画布
- **那么** 画布视图按拖拽方向移动，显示图的其他部分

#### 场景:适应画布显示所有节点
- **当** 用户点击"适应画布"或类似功能按钮
- **那么** 画布自动调整缩放级别和平移位置，使所有节点都在可视区域内
